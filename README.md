**ARI711S – Artificial Intelligence Assignment 1**

This repository contains our group's solutions for the ARI711S Artificial Intelligence assignment, developed as part of the Bachelor of Computer Science (Software Development) programme at the Namibia University of Science and Technology (NUST).

The project covers four core AI topics implemented in Python:

- **Question 1** – Informed search algorithms (Greedy Best-First Search and A*) applied to a warehouse robot navigation problem
- **Question 2** – Constraint Satisfaction Problem (CSP) solver using backtracking, MRV heuristic and Forward Checking for 5G telecommunication tower placement
- **Question 3** – Adversarial search using the Minimax algorithm with alpha-beta pruning to build an unbeatable Tic-Tac-Toe AI
- **Question 4** – Reinforcement Learning using the SARSA algorithm to find the optimal value function and policy for a Gridworld MDP

All solutions are documented in a single Jupyter notebook with Markdown explanations, visual outputs, and test cases.

**Technologies used:** Python, Matplotlib, NumPy, Tkinter
